To run project Items Catalog: 
1. Navigate to the Full-Stack-Foundations/Lesson-4/Final-Project directory inside 
In Command Prompt:
2. Run "database_setup.py"  to create database


3. Run "items.py" to populate the database with items of catalog

4. Run ItemsCatalog.py and navigate to localhost:5000 in your browser
5. In your browser now you can use web-app, which is online catalog of items. 
   You can see name, description and price of the item. To change, add or delete
   item you need to log in, by using your Google Account.
